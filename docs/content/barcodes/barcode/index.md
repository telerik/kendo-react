# KendoReact Barcode Overview

The KendoReact Barcode renders a variety of one-dimensional (1D) industry barcodes.

The KendoReact Barcode is distributed through the [kendo-react-barcodes NPM package](https://www.npmjs.com/package/@progress/kendo-react-barcodes).

> The Barcode is part of the [KendoReact Barcodes]({% slug overview_barcodes %}) component library. The procedures for installing, importing, and using the Barcodes are identical for all components in the package. To learn how to use the Barcode and the rest of the Barcodes, see the [Getting Started with the KendoReact Barcodes]({% slug get_started_barcodes %}) guide.

## Key Features

-   [Barcode types]({% slug types_barcode_barcodes %})&mdash;You can use an extensive set of encoding schemes (symbologies) that are supported by the Barcode.
-   [Configuration options]({% slug configuration_barcode_barcodes %})&mdash;The Barcode provides fine-grained control over its behavior and appearance.
-   [Rendering modes]({% slug rendering_barcodes %})&mdash;You can also display the Barcode in a Canvas (bitmap) or an SVG (vector graphics) format.
-   [Export options]({% slug export_barcodes %})&mdash;The Barcode enables you to export its contents to an Image (PNG), PDF, and SVG format.
