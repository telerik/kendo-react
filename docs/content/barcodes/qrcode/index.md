# KendoReact QR Code Overview

The KendoReact QR Code supports the popular Quick Response codes with a variety of settings.

The KendoReact QR Code is distributed through the [kendo-react-barcodes NPM package](https://www.npmjs.com/package/@progress/kendo-react-barcodes).

> The QR Code is part of the [KendoReact Barcodes]({% slug overview_barcodes %}) component library. The procedures for installing, importing, and using the Barcodes are identical for all components in the package. To learn how to use the QR Code and the rest of the Barcodes, see the [Getting Started with the KendoReact Barcodes]({% slug get_started_barcodes %}) guide.

## Key Features

-   [Encoding and error correction]({% slug encoding_qrcode_barcodes %})&mdash;The QR Code supports different encodings and error correction levels.
-   [Overlays]({% slug overlays_qrcode_barcodes %})&mdash;The QR Code comes with built-in support for Swiss codes and image overlays.
-   [Configuration options]({% slug configuration_qrcode_barcodes %})&mdash;The QR Code provides fine-grained control over it's behavior and appearance.
-   [Rendering modes]({% slug rendering_barcodes %})&mdash;You can also display the QR Code in a Canvas (bitmap) or an SVG (vector graphics) format.
-   [Export options]({% slug export_barcodes %})&mdash;The QR Code enables you to export its contents to an Image (PNG), PDF, and SVG format.
