# FloatingActionButtonRounded
Specifies the rounding of the Floating Action Button.

The possible values are:
* `small`
* `medium`
* `large`
* `full`
* `none`

`type` FloatingActionButtonRounded = "small" | "medium" | "large" | "full" | "none";
