# Toggleable Button

The Button enables you to indicate whether it is active or inactive. A toggleable button is useful when users need to switch between two states, such as enabling or disabling a feature, turning settings on or off, or activating modes in an application.

To make a Button toggleable, update the default `false` configuration of the [`togglable`]({% slug api_buttons_buttonprops %}#toc-togglable) property and set it to `true`.
