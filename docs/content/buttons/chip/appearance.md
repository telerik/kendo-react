# Appearance

The Chip provides a predefined set of appearance options.

Apart from the default vision of the Chip, these alternative styling options enable you to configure each individual aspect of the appearance of a Chip.

-   [Size](#chip-size)
-   [Border Radius](#chip-border-radius)
-   [Fill Mode](#chip-fill-mode)
-   [Theme Colors](#chip-theme-colors)

## Chip Size

The Chip provides the [`size`]({% slug api_buttons_chipprops %}#toc-size) property that enables you to control how big or small the rendered Chip will be.
