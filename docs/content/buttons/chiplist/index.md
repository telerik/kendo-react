# KendoReact ChipList Overview

The ChipList allows you to organize a number of individual chips in a manageable list.

The KendoReact ChipList is distributed through the [kendo-react-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-react-buttons).

## Key Features

-   [Selection Mode]({% slug selectionmode_chiplist %})&mdash;You can control the selection mode and choose between the `single`, `multiple`, or `none` modes.
-   [Controlled Mode]({% slug controlledmode_chiplist %})&mdash;You can configure the ChipList for a controlled mode.
-   [Data Binding]({% slug databinding_chiplist %})&mdash;The ChipList allows you to bind your own data.
-   [Customization]({% slug customization_chiplist %})&mdash;The ChipList is customizable and you can style it according to your requirements.
-   [Keyboard Navigation]({% slug keyboard_navigation_chiplist %})&mdash;The ChipList supports various keyboard shortcuts.
-   [Accessibility]({% slug accessibility_chiplist %})&mdash;The ChipList is accessible for screen readers and supports WAI-ARIA attributes.
-   [Disabled ChipList]({% slug disabled_chiplist %})&mdash;You can easily render a disabled ChipList and prevent end-users from interacting with it.
