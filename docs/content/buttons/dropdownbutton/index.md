# KendoReact DropDownButton Overview

The KendoReact DropDownButton displays a popup list with action items and provides options for enabling or disabling its content, displaying icons, and binding it to data.

The KendoReact DropDownButton is distributed through the [kendo-react-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-react-buttons).

## Key Features

-   [Disabled DropDownButton]({% slug disabled_dropdownbutton %})&mdash;You can easily render a disabled DropDownButton and prevent end-users from interacting with it.
-   [Icon DropDownButton]({% slug icons_dropdownbutton %})&mdash;The DropDownButton allows you to display images, predefined or custom icons.
-   [Popup and items customization]({% slug customization_dropdownbutton %})&mdash;You can customize the rendering and behavior of the popup and its items.
-   [Data binding]({% slug binding_dropdownbutton %})&mdash;You can bind the DropDownButton to various data types.
-   [Globalization]({% slug rtl_buttons %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
-   [Keyboard navigation]({% slug keyboard_navigation_dropdownbutton %})&mdash;The DropDownButton supports various keyboard shortcuts.
-   [Accessibility]({% slug accessibility_dropdownbutton %})&mdash;The DropDownButton is accessible for screen readers and supports WAI-ARIA attributes.
