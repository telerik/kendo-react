# KendoReact FloatingActionButton Overview

The FloatingActionButton is used to specify the primary or the most common action in an application.

The KendoReact FloatingActionButton is distributed through the [kendo-react-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-react-buttons).

## Key Features

-   [Disabled FloatingActionButton]({% slug disabled_floatingactionbutton %})&mdash;You can easily render a disabled FloatingActionButton and prevent end-users from interacting with it.
-   [Content Types]({% slug contenttypes_floatingactionbutton %})&mdash;The FloatingActionButton allows you to specify the visual content of the rendered button.
-   [Positioning]({% slug positioning_floatingactionbutton %})&mdash;You can take advantage of various positioning options.
-   [Appearance]({% slug appearance_floatingactionbutton %})&mdash;The FloatingActionButton comes with many predefined shapes, sizes, and theme colors.
-   [Items]({% slug databinding_floatingactionbutton %})&mdash;You can customize the FloatingActionButton items through the available configuration options.
-   [Keyboard Navigation]({% slug keyboard_navigation_floatingactionbutton %})&mdash;The FloatingActionButton supports various keyboard shortcuts.
-   [Accessibility]({% slug accessibility_floatingactionbutton %})&mdash;The FloatingActionButton is accessible for screen readers and supports WAI-ARIA attributes.
