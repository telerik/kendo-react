# Customization

The SegmentedControl allows you to fully customize the visual appearance of each item by using the [`itemTemplate`](slug:api_buttons_segmentedcontrolprops#itemTemplate) prop.

The `itemTemplate` is a function that receives the item's data ([`SegmentedItemProps`](slug:api_buttons_segmenteditemprops)) and returns a React node to render inside the item button.

The following demo shows how to use a custom template to render each item with an icon, a label, and a badge element.
