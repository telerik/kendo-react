# Disabled Segments

The SegmentedControl allows you to disable individual items by setting the `disabled` property on the corresponding item in the [`items`]({% slug api_buttons_segmentedcontrolprops %}#toc-items) collection. A disabled segment cannot be selected or interacted with.
