# Layout Mode

The SegmentedControl provides the [`layoutMode`](slug:api_buttons_segmentedcontrolprops#layoutMode) property that controls how items are distributed horizontally.

The available layout modes are:

-   `compact`&mdash;Items are sized based on their content. This is the default.
-   `stretch`&mdash;Items stretch to fill the available horizontal space equally.
