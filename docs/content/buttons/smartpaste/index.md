# KendoReact SmartPasteButton Overview

The KendoReact SmartPasteButton is a specialized button component that leverages AI to intelligently fill form fields from clipboard content. It combines standard button properties with AI-powered form population, making it easy to extract structured data from unstructured text.

The SmartPasteButton component analyzes clipboard content, understands the form structure, and automatically maps data to the appropriate fields, significantly improving data entry efficiency and user experience.

The KendoReact SmartPasteButton is distributed through the [kendo-react-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-react-buttons).
