# Keyboard Navigation

## Managing the Focus

The SmartPasteButton is focusable by default, and follows the regular tab order of the page.

## Keyboard Navigation

The SmartPasteButton supports the following keyboard shortcuts:

| Shortcut           | Behavior                                         |
| ------------------ | ------------------------------------------------ |
| `Enter` or `Space` | Triggers a click action on the SmartPasteButton. |
