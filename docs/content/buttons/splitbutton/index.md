# KendoReact SplitButton Overview

The KendoReact SplitButton allows the user to execute a default action that is bound to a Button or to choose a predefined action from a drop-down list.

The KendoReact SplitButton is distributed through the [kendo-react-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-react-buttons). See the [package overview]({% slug overview_buttons %}) for more details.
