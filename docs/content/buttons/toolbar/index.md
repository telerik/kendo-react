# KendoReact Toolbar Overview

The KendoReact Toolbar represents a graphical element for holding button elements and enables the user to easily structure and access them.

The KendoReact Toolbar is distributed through the [kendo-react-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-react-buttons). See the [package overview]({% slug overview_buttons %}) for more details.

## Key Features

-   [Tools]({% slug content_toolbar %})&mdash;You can include various button types in the Toolbar and create a handy selection of tools.
-   [Adaptive Rendering]({% slug adaptive_rendering_toolbar %})&mdash;The ToolBar provides options to hide the tools that do not fit its width in an overflow section or rendering scroll arrows.
-   [Globalization]({% slug rtl_buttons %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
-   [Keyboard Navigation]({% slug keyboard_navigation_toolbar %})&mdash;The Toolbar supports various keyboard shortcuts.
-   [Accessibility]({% slug accessibility_toolbar %})&mdash;The Toolbar is accessible for screen readers and supports WAI-ARIA attributes.
