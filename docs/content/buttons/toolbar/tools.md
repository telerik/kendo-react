# Tools

The KendoReact Toolbar allows you to render Button, ButtonGroup, DropDownButton, SplitButton and custom button components in it.

To include components in the Toolbar, wrap each of them into a [ToolbarItem]({% slug api_buttons_toolbaritem %}). You can also separate them from each other with the [ToolbarSeparator]({% slug api_buttons_toolbarseparator %}) or [ToolbarSpacer]({% slug api_buttons_toolbarspacer %}).
