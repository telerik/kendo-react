# createState
Creates a state object from the provided data and series type.




#### Parameters
##### data
<code>


[ChartWizardDataRow]({% slug api_chart-wizard_chartwizarddatarow %})[]


</code>
Specifies the data you want to display in the ChartWizard component.

##### seriesType
<code>


[ChartWizardSeriesType]({% slug api_chart-wizard_chartwizardseriestype %})


</code>
Sets the series type of the ChartWizard component.

#### Returns
<code>


[ChartWizardState]({% slug api_chart-wizard_chartwizardstate %})


</code>
 The state object.
