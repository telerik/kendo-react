# KendoReact Chart Wizard Overview

The KendoReact Chart Wizard lets you create a chart using data from a Grid, another data-bound component, or an external source. The Chart Wizard has a simple interface that allows you to set up the chart type, series, category axis, value axis, legend, area, title, and data source. The Chart Wizard is available under the [@progress/kendo-react-chart-wizard NPM package](https://www.npmjs.com/package/@progress/kendo-react-chart-wizard).

## React Chart Wizard Key Features

The KendoReact Chart Wizard component is a powerful tool that allows you to create a chart with a few clicks. The wizard is designed to be easy to use and intuitive, so you can create and save charts quickly and easily. The KendoReact team continuously invests efforts to improve the performance and add more value to the existing Chart Wizard library as well as develop new features to it.

### Data Binding

The Chart Wizard can be bound to data collections and comes with built-in integrations with the Grid component.
[Read more about the data binding options of the Chart Wizard...](slug:databinding_chart-wizard)

### Export Options

The Chart Wizard allows the user to save the charts they created in PDF, PNG and SVG formats.
[Read more about the data export options of the Chart Wizard...](slug:export_chart-wizard)
