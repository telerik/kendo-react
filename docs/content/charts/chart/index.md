## Key Features

-   [Data binding]({% slug databinding_chart_charts %})&mdash;You can bind the Chart's data series and category axis to arrays and datasets of objects.
-   [Rendering modes]({% slug rendering_chart_charts %})&mdash;You can configure the Chart to render its graphs as Canvas (bitmap) or SVG (vector graphics).
-   [Export options]({% slug export_chart_charts %})&mdash;The Chart allows end-users to export the graphs as image, PDF, and SVG files.
-   [Panning and zooming]({% slug panzoom_chart_charts %})&mdash;End users can intuitively focus on specific chart ranges by panning and zooming the Chart component.
-   [Appearance]({% slug appearance_chart_charts %})&mdash;You can use various configuration options to set the Chart's dimensions and adjust the displayed range. Additionally, you can choose between the three gorgeous KendoReact themes.
-   [Globalization]({% slug globalization_charts %})&mdash;The Chart has built-in globalization features allowing you to create applications that can be used all over the world.
-   [KendoReact Chart Events]({% slug events_chart %})&mdash;The variety of events that the Chart generates allow you to customize the component's behavior.
