# KendoReact StockChart Overview

The KendoReact StockChart is a specialized control for visualizing the price movement of a financial instrument over a certain period of time.

Generally, Stockcharts include extensive touch support and a navigator pane for easy browsing of extended time periods. Generally, the StockChart extends the [KendoReact Chart]({% slug overview_chart_charts %}) component and shares most of its features.

## Key Features

-   [Navigator]({% slug navigator_stockchart_charts %})&mdash;You can use the navigator pane of the StockChart to change the date interval of the main panes.
-   The StockChart also supports all configuration options of the [Chart]({% slug api_charts_chartprops %}).
