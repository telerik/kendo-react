# Material UI

KendoReact enables you to integrate the components with [Material UI](https://material-ui.com/).

The [KendoReact Material theme](https://www.telerik.com/design-system/docs/themes/kendo-themes/material/) fully corresponds to the Material UI guidelines and allows you to add KendoReact components to an already existing application which contains Material UI components. You can also add Material UI components and integrate them with KendoReact for a specific purpose&mdash;for example, as editors in the Grid.

## Dashboard Application

The Dashboard application demonstrates how to use KendoReact components and Material UI components, and follows the Material UI guidelines.

-   To access the live project, refer to [this GitHub repository](https://telerik.github.io/kendo-react-material-dashboard/#/dashboard).
-   To download the project, refer to [this GitHub repository](https://github.com/telerik/kendo-react-material-dashboard).
