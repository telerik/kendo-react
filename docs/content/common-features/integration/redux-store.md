# Redux Store

KendoReact enables you to bind the components to [Redux](https://redux.js.org/basics/usage-with-react) by using the state.

The components provide events which can be used to dispatch [actions](https://redux.js.org/basics/actions) to the [reducers](https://redux.js.org/basics/reducers), and to update the [store](https://redux.js.org/basics/store).

For more information on how to bind multiple components to the Redux store and how the update of a component affects the other components that are bound to the same state, refer to the [KendoReact Dashboard sample application](https://telerik.github.io/kendo-react-material-dashboard/). The source code of the application is located in [this GitHub repository](https://github.com/telerik/kendo-react-material-dashboard).
