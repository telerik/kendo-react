# dispatchKendoPasteEvent
Dispatches a smart paste populate event from the given element.
The event will bubble up through the DOM tree.




#### Parameters
##### element
<code>


"null" | Element


</code>
The element to dispatch the event from

##### fieldValues
<code>


Record&lt;string&gt;


</code>
The field values to populate

#### Returns
<code>


boolean


</code>
 true if the event was not cancelled, false otherwise
