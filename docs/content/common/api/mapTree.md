# mapTree
Creates a new array with the results of calling the provided callback function
on every element in the provided data tree.




#### Parameters
##### tree
<code>


any[]


</code>
The data tree.

##### subItemsField
<code>


string


</code>
The field which points to the subitems collection of each data item.

##### callback
<code>


(value: any) => any


</code>
The callback function.

#### Returns
<code>


any[]


</code>
 - The new data tree.
