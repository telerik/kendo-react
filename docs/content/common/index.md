## How Do the KendoReact Common Utilities Work?

> The KendoReact Common Utilities provide both [free React common utilities](#free-react-common-utilities) and premium common utilities which require a commercial license key or an active trial license key. If you need to develop apps using the [premium React common utilities](#premium-react-common-utilities) or other premium KendoReact components and features, follow the instructions on the [My License page](slug://my_license) to activate your license.

### React Common Utilities with Other KendoReact Components

You can combine multiple KendoReact components with the React common utilities in the same application:

-   Use the Draggable component to enable drag-and-drop functionality for [Cards](slug://overview_card) and other KendoReact components, creating a more interactive UI.
-   Add Icons or SvgIcons to your [Buttons](slug://overview_buttons).
-   Use AdaptiveModeContext to configure adaptive rendering for components such as [DatePicker](slug://overview_datepicker), [DropDownList](slug://overview_dropdownlist), [ComboBox](slug://overview_combobox) and many more.
-   Configure Keyboard Navigation for your [ButtonGroup](slug://overview_buttongroup).

> The Common Utilities package is a dependency for multiple other KendoReact components.

## Frequently Asked Questions

### Are the KendoReact Common Utilities Free to Use?

The KendoReact Common Utilities provide both [free React common utilities](#free-react-common-utilities) and [premium common utilities](#premium-react-common-utilities) which require a commercial license key or an active trial license key.

You can start using the free components without any license or registration with Progress or Telerik.

### How to Upgrade to the Premium Common Utilities?

If you want to use any of the premium [React common utilities](#premium-react-common-utilities), UI components, or component features, you need a valid commercial license or an active trial license. A [free trial](https://www.telerik.com/try/kendo-react-ui) is available, if you want to try out the premium features before purchase.

### What are the Available Support Options?

For any questions about the use of KendoReact Common Utilities, or any other [KendoReact components](https://www.telerik.com/kendo-react-ui/components/), there are [several free support options available](https://www.telerik.com/kendo-react-ui/support):

-   The [KendoReact forums](https://www.telerik.com/forums/kendo-ui-react) are part of the free support you can get from the community and from the KendoReact team on all kinds of general issues.
-   [KendoReact Feedback Portal](https://feedback.telerik.com/kendo-react-ui) and [KendoReact Roadmap](https://www.telerik.com/support/whats-new/kendo-react-ui/roadmap/) provide information on the features in discussion and also those planned for release.
-   KendoReact uses [GitHub Issues](https://github.com/telerik/kendo-react) as its bug tracker, and you can submit any related reports there. Also, check out the [closed list](https://github.com/telerik/kendo-react/issues?q=is%3Aissue+is%3Aclosed).

**KendoReact license holders and anyone in an active trial** can take advantage of the outstanding KendoReact customer support delivered by the developers who built the library. To submit a support ticket, use the [Telerik support system](https://www.telerik.com/account/support-tickets).

Need something unique that is tailor-made for your project? Progress offers its [Progress Services](https://www.progress.com/services) group that can work with you to create any customized solution that you might need.
