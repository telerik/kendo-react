# KendoReact AIPrompt Overview

The KendoReact AIPrompt enables you to write prompts and execute predefined commands as well as look and interact with the output from the AI.

The KendoReact AIPrompt component is distributed through the [kendo-react-conversational-ui npm package](https://www.npmjs.com/package/@progress/kendo-react-conversational-ui).

## Key Features

-   [Suggestions](slug:suggestions_aiprompt)—The KendoReact AIPrompt provides the ability to create suggestions.
-   [Streaming](slug:streaming_aiprompt)—The KendoReact AIPrompt supports streaming responses from the AI in real time.
-   [Customization](slug:customization_aiprompt)—The KendoReact AIPrompt provides the ability to customize components.
-   [Commands](slug:commands_aiprompt)—The KendoReact AIPrompt provides the ability to create custom commands.
-   [Events](slug:events_aiprompt)—The KendoReact AIPrompt exposes different events which enable you to control its behavior upon user interaction.
