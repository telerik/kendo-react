# AttachmentLayout
Specifies the layout for message attachments.

The supported values are:
* `"list"`&mdash;A vertical list.
* `"carousel"`&mdash;A horizontal, scrollable list. Also called a card deck.

`type` AttachmentLayout = "list" | "carousel";
