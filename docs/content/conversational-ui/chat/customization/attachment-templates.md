# Attachment Templates

The Chat allows you to customize how file attachments are displayed through the [`attachmentTemplate`](slug:api_conversational-ui_chatprops#toc-attachmenttemplate) property.

To set a custom attachment template, create a React component and assign it to the `attachmentTemplate` property of the Chat component.
