# Inline AI Prompt Commands

The KendoReact Inline AI Prompt enables you to define custom prompt commands and output actions

The following demo demonstrates how to use the KendoReact Inline AI Prompt with custom commands. Select some text in the textarea to trigger the inline AI prompt. Click the icon with the three horizontal lines to view available commands. After choosing a command, you can copy the result, insert it into the textarea, replace the selected text, or discard the output.
