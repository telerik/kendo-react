# Customization

The KendoReact Inline AI Prompt enables you to render custom components for the Generate button and custom output cards. This can be achieved by passing a custom component to the [`generateButton`]({% slug api_conversational-ui_inlineaipromptprops %}#toc-generateButton) and [`outputCard`]({% slug api_conversational-ui_inlineaipromptprops %}#toc-outputCard) props.
