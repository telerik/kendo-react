# KendoReact Inline AI Prompt Overview

The KendoReact Inline AI Prompt enables you to write prompts and interact with AI-generated content through a flexible popup interface that can be positioned relative to any element in your application.

The KendoReact Inline AI Prompt component is distributed through the [kendo-react-conversational-ui npm package](https://www.npmjs.com/package/@progress/kendo-react-conversational-ui).
