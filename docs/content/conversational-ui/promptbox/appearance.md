# PromptBox Appearance

The PromptBox provides predefined appearance options such as different sizes, border radiuses, fill modes and theme colors for its built-in button controls.

The following example demonstrates all available appearance options of the PromptBox in action.

<demo metaUrl="conversational-ui/promptbox/appearance/overview/" height="340"></demo>

## Size

The PromptBox allows you to configure the `padding` of its buttons. To achieve this utilize the [`size`](slug:api_buttons_buttonprops#size) property for each button.

The following example demonstrates how to define the size of the PromptBox buttons.

<demo metaUrl="conversational-ui/promptbox/appearance/size/" height="300"></demo>

## Roundness

The PromptBox enables you to apply different `border radius` to its buttons through the [`rounded`](slug:api_buttons_buttonprops#rounded) property.

The following example demonstrates how to define the border radius of the PromptBox buttons.

<demo metaUrl="conversational-ui/promptbox/appearance/rounded/" height="310"></demo>

## Fill Mode

The PromptBox enables you to set different fill modes for its buttons by using the [`fillMode`](slug:api_buttons_buttonprops#fillMode) property.

The following example demonstrates how to define the fill mode of the PromptBox buttons.

<demo metaUrl="conversational-ui/promptbox/appearance/fillmode/" height="400"></demo>

## Theme Colors

The PromptBox enables you to set different theme colors for its buttons by using the [`themeColor`](slug:api_buttons_buttonprops#themeColor) property.
