# getItemPath
Returns the data item path in the tree based on the level parameter.




#### Parameters
##### tree
<code>


any[]


</code>
The data tree.

##### level
<code>


number[]


</code>
The level of the target tree item.

##### subItemsField?
<code>


string


</code>
The field which points to the subitems collection of each data item.

#### Returns
<code>


any[]


</code>
 - The path of the data item.
