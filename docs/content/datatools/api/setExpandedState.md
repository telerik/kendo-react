# setExpandedState
Apply the `expanded` prop to the group items in data based on the provided collection of group ids.




#### Parameters
##### options
<code>


{ collapsedIds: string[]; data: any; }


</code>
The options to be processed.

#### Returns
<code>


any[]


</code>
 - Collection of all group ids from the data.
