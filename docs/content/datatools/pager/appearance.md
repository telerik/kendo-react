# Appearance

The KendoReact Pager allows selecting a predefined size option that changes its size.

To change the default size of the Pager, set its [`size`]({% slug api_data-tools_pagerprops %}#toc-size) property which will change the padding of its elements. The available size options are:

-   `small` - sets the padding of the elements to 2px 8px
-   `medium` (default) - sets the padding of the elements to 4px 8px
-   `large` - sets the padding of the elements to 8px 8px
