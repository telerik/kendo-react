# Disabled Calendar

By default, the Calendar is enabled.

You can disable the Calendar by setting its [`disabled`]({% slug api_dateinputs_calendarprops %}#toc-disabled) property. When `disabled` is set to `true`, date selection is disabled and the Calendar renders with different styles.
