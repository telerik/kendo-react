# Week Number Column

By default, the Calendar does not render a week number column.

You can render a week number column by setting the [`weekNumber`]({% slug api_dateinputs_calendarprops %}#toc-weeknumber) property. When set to `true`, a column which displays the numbers of the weeks is rendered.
