# Disabled DateInput

By default, the DateInput is enabled.

You can disable the DateInput by setting its [`disabled`]({% slug api_dateinputs_dateinputprops %}#toc-disabled) property. When set to `true`, date selection is disabled and different styles are applied to the DateInput.
