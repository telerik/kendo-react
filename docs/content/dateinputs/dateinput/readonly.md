# Read-Only DateInput

By default, the DateInput is in an active state.

You can enable the read-only state of the DateInput by setting its [`readonly`]({% slug api_dateinputs_dateinputprops %}#toc-readonly) property. When set to `true`, date selection is disabled, the styles applied to the DateInput remain the same.
