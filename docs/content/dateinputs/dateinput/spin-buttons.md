# Spin Buttons

By default, the spin buttons are disabled.

You can enable the spin buttons of the DateInput by setting its [`spinners`]({% slug api_dateinputs_dateinputprops %}#toc-spinners) property. When set to `true`, spin buttons are rendered inside of the DateInput.
