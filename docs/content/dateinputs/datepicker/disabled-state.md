# Disabled DatePicker

By default, the DatePicker is enabled.

You can disable the DatePicker by setting its [`disabled`](slug:api_dateinputs_datepickerprops#disabled) property. When set to `true`, date selection is disabled and different styles are applied to the DatePicker.
