# Focused Dates

The DateTimePicker enables you to handle the configuration of the focused calendar dates.

The Calendar always displays a focused date and, by default, the focused date is today's date. To change the focused date, use the [`focusedDate`]({% slug api_dateinputs_datetimepickerprops %}#toc-focuseddate) property by setting it to a specific `Date` value. The `Date` value has to be a valid JavaScript `Date` object.
