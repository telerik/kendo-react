# Disabled DateTimePicker

By default, the DateTimePicker is enabled.

You can disable the DateTimePicker by setting its [`disabled`](slug:api_dateinputs_datetimepickerprops#disabled) property. When set to `true`, date selection is disabled and different styles are applied to the DateTimePicker.
