# Multiple Views

The MultiViewCalendar enables you to control the number of the horizontally rendered views.

To set the view number, use the [`views`]({% slug api_dateinputs_multiviewcalendarprops %}#toc-views) property. By default, the MultiViewCalendar renders two views.
