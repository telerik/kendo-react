# Action Buttons

The Dialog provides options for rendering action buttons and customizing their content.

To specify action buttons in the Dialog, use the `DialogActionsBar` element. You can customize the layout of the buttons by using the [`layout`]({% slug api_dialogs_dialogactionsbarprops %}#toc-layout) property of the `DialogActionsBar` component.
