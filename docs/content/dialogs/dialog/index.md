# KendoReact Dialog Overview

The KendoReact Dialog communicates specific information and prompts users to take specific actions by interacting with a modal dialog.

The KendoReact Dialog is distributed through the [kendo-react-dialogs NPM package](https://www.npmjs.com/package/@progress/kendo-react-dialogs).

To learn how to use the Dialog and the rest of the Dialogs, see the [Getting Started with the KendoReact Dialogs]({% slug get_started_dialogs %}) guide.

## Key Features

-   [Title]({% slug title_dialog %})&mdash;You can customize the title of the Dialog.
-   [Visibility]({% slug visibilitty_dialog %})&mdash;You can control when the Dialog is visible on the page.
-   [Action buttons]({% slug actionbuttons_dialog %})&mdash;You can add different actions buttons to the Dialog.
-   [Dimensions]({% slug dimensions_dialog %})&mdash;You can set the Dialog's initial `width` and `height`.
-   [Globalization]({% slug globalization_dialog %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
-   [Keyboard navigation]({% slug keyboard_navigation_dialog %})&mdash;The Dialog supports various keyboard shortcuts.
