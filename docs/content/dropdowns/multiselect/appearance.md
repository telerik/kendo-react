# Appearance

The MultiSelect provides a predefined set of appearance options.

Apart from the default vision of the MultiSelect, these alternative styling options enable you to configure each individual aspect of the appearance of a MultiSelect.

## Size

You can set the size of the MultiSelect through the [`size`]({% slug api_dropdowns_multiselectprops %}#toc-size) property. The MultiSelect component comes in three different sizes: `small`, `medium` (default), and `large`.

## Fill Mode

You can set the fill mode of the MultiSelect through the [`fillMode`]({% slug api_dropdowns_multiselectprops %}#toc-fillmode) property. The MultiSelect provides three different fill mode options: `solid` (default), `outline`, and `flat`.

## Rounded

You can set the roundness of the MultiSelect through the [`rounded`]({% slug api_dropdowns_multiselectprops %}#toc-rounded) property. The MultiSelect component supports five different degrees of roundness: `small`, `medium` (default), `large`, `full`, and `none`.
