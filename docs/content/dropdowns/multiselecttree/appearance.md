# Appearance

The MultiSelectTree provides a predefined set of appearance options.

Apart from the default vision of the MultiSelectTree, these alternative styling options enable you to configure each individual aspect of the appearance of a MultiSelectTree.

## Size

You can set the size of the MultiSelectTree through the [`size`]({% slug api_dropdowns_multiselecttreeprops %}#toc-size) property. The MultiSelectTree component comes in three different sizes: `small`, `medium` (default), and `large`.

## Fill Mode

You can set the fill mode of the MultiSelectTree through the [`fillMode`]({% slug api_dropdowns_multiselecttreeprops %}#toc-fillmode) property. The MultiSelectTree provides three different fill mode options: `solid` (default), `outline`, and `flat`.

## Rounded

You can set the roundness of the MultiSelectTree through the [`rounded`]({% slug api_dropdowns_multiselecttreeprops %}#toc-rounded) property. The MultiSelectTree component supports five different degrees of roundness: `small`, `medium` (default), `large`, `full`, and `none`.
