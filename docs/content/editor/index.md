# React Rich Text Editor Overview

The React Editor enables users to create rich textual content through a What-You-See-Is-What-You-Get (WYSIWYG) interface and delivers a set of tools for creating, editing, and formatting of text, paragraphs, lists, and other HTML elements.

<CtaPanelOverview></CtaPanelOverview>

The React react rich text Editor, part of KendoReact, is built from the ground up and specifically for React, so that you get a high-performance control which integrates tightly with your application and with the rest of the KendoReact components. The KendoReact Editor component is distributed through the [kendo-react-editor NPM package](https://www.npmjs.com/package/@progress/kendo-react-editor).

## Key Features of the React Text Editor

The KendoReact Editor component delivers a range of handy developer-friendly features whose number and further development are not limited by the list in this section. The KendoReact team constantly invests efforts to improve the performance, add more value to the existing Editor library, and develop new features.

-   [Tools]({% slug tools_editor %})&mdash;A set of built-in, user-interface tools, that enables you to customize the available tools and to add custom ones.
-   [Getting/Setting HTML Content]({% slug content_editor %})&mdash;You can programmatically get the HTML content of the KendoReact Editor and set an updated version back.
-   [Controlled Mode]({% slug controlled_editor %})&mdash;Takes advantage of the built-in options that allow you to control the value of the Editor.
-   [Rendering Modes]({% slug renderingmode_editor %})&mdash;Allows you to render the KendoReact Editor content in a div element. By default the Editor render its content in an iframe element.
-   [Custom Rendering]({% slug customrendering_editor %})&mdash;Provides options for customizing the way the Editor renders its elements.
-   [Appearance]({% slug styling_editor %})&mdash;Allows you to add custom styles for the KendoReact Editor's content.
-   [Sanitizing Pasted HTML Content]({% slug paste_editor %})&mdash;The KendoReact Editor exposes a set of built-in functions for sanitizing the content which the user pastes in the content area.
-   [Plugins]({% slug plugins_editor %})&mdash;Allows you to extend and customize the Editors functionality and features by using plugins.
-   [Modifying the Schema]({% slug schema_editor %})&mdash;By modifying the default, built-in schema, you can add additional types of nodes and marks as well as edit and remove the existing ones.
-   [Selection]({% slug selection_editor %})&mdash;The KendoReact Editor component supports several types of selection: TextSelection, NodeSelection, CellSelection, and AllSelection.
-   [Resizing]({% slug resizing_editor %})&mdash;The KendoReact Editor component provides a resizing feature. You can also allow the resizing of images by using the provided built-in plugin.
-   [Find and Replace]({% slug findandreplace_editor %})&mdash;Provides you built-in tools for find and replace text content.
-   [Images]({% slug images_editor %})&mdash;Allows you to paste, drag and drop, and upload images. You can implement it and extend the built-in functionality.
-   [Keyboard Navigation]({% slug keyboard_navigation_editor %})&mdash;Keyboard navigation functionality for the KendoReact Editor that is enabled by default.
-   [Globalization]({% slug globalization_editor %})&mdash;The globalization process combines the translation of component messages with adapting them to specific locales.

## Toolbar Tools

The React Editor, part of KendoReact, uses the KendoReact ToolBar component to deliver UI tools and directives for associating the toolbar tools with edit-action commands. [Read more about the toolbar tools of the Editor...]({% slug tools_editor %})

## React Rich Text Editor Appearance

You can use the React rich text Editor, part of KendoReact, options for setting its style mode, defining the height of its content as well as customizing its appearance altogether. [Read more about the styling options of the Editor...]({% slug styling_editor %})

## Globalization

The React Editor, part of KendoReact, supports globalization to ensure that it can fit well in any application, no matter what languages and locales need to be supported. Additionally, the Editor supports rendering in a right-to-left (RTL) direction. [Read more about the globalization of the Editor...]({% slug globalization_editor %})

## Keyboard Navigation

The React Editor, part of KendoReact, supports a number of keyboard shortcuts which allow users to accomplish various commands. [Read more about the keyboard navigation of the Editor...]({% slug keyboard_navigation_editor %})
