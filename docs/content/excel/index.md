## Known Limitations

-   Older browsers, such as Internet Explorer 9 and Safari, require you to implement a server proxy. For more information, refer to the section on the [`proxyUrl`]({% slug api_excel-export_excelexportprops %}#toc-proxyurl) configuration.
-   The maximum size of the exported file has a system-specific limit. For large data sets, it is highly recommended that you use a server-side solution.
