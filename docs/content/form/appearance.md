# Appearance

The KendoReact FormElement allows selecting a predefined size option that changes its size.

## Size

The `size` property controls the overall size of the form element, and the spacing within the form. The available size options are:

-   `small`
-   `medium` (default)
-   `large`

To change the default size of the FormElement, set its [`size`]({% slug api_form_formelementprops %}#toc-size) property.
