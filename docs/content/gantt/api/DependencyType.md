# DependencyType
The dependency type when two tasks are connected.

The supported values are:
* `FF`&mdash;from 'finish' to 'finish'
* `FS`&mdash;from 'finish' to 'start'
* `SS`&mdash;from 'start' to 'start'
* `SF`&mdash;from 'start' to 'finish'
