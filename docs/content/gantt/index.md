# React Gantt Overview

The React Gantt displays hierarchical tasks data in tabular and timeline formats. It comes with a comprehensive set of ready-to-use features covering everything from data binding directives, sorting, filtering, tasks, and dependencies to intuitive display of task timelines, progression, and styling.

The Gantt is built from the ground up and specifically for React, so that you get a high-performance control which integrates tightly with your application and with the rest of the KendoReact components.

The Kendo React Gantt component is distributed through the [kendo-react-gantt NPM package](https://www.npmjs.com/package/@progress/kendo-react-gantt).

## React Gantt Key Features

The Kendo React Gantt component delivers a range of handy and developer-friendly features whose number and further development are not limited by the list in this section. The KendoReact team constantly invests efforts to improve the performance, add more value to the existing Gantt library, and develop new features.

### Sorting

The Gantt supports a number of sorting options including sorting and unsorting single and multiple columns as well as pre-sorting its data records. [Read more about the sorting options of the Gantt...]({% slug sorting_gantt %})

### Filtering

The Gantt provides options for displaying only those data records which meet specified criteria and also for additionally customizing its filtering behavior. [Read more about filtering the Gantt records...]({% slug filtering_gantt %})

### Tasks

The Gantt renders a task element for each of its expanded data item rows, automatically displays the relevant dependencies between the tasks, and provides task templates for showing additional information and custom content for each Gantt task. [Read more about setting the task options of the Gantt...]({% slug task_types_gantt %})

### Columns

The Gantt supports extensive configuration settings for controlling the display and behavior of its columns, such as hiding and locking them, spanning their row content, applying reordering and real-time resizing, and also using templates for their customization. [Read more about the configuring the columns in the Gantt...]({% slug reordering_columns_gantt %})

### Editing

The Gantt supports creating, updating, and deleting its records. [Read more about the editing of the Gantt...]({% slug editing_gantt %})
