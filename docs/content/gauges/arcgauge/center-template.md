# Center Template

You can customize the content inside the center of the ArcGauge by using the center template.

To render the center template of an ArcGauge, pass a function to the [`centerRender`]({% slug api_gauges_arcgaugeprops %}#toc-centerRender) property which accepts value and color arguments.
