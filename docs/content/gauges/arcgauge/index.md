# KendoReact ArcGauge Overview

The KendoReact ArcGauge represents value ranges on its arc.

Compared to the [Radial Gauge]({% slug overview_radialgauge_gauges %}), this gauge uses a colored arc instead of an arrow pointer. This leaves the center free for a user-defined [Center Template]({% slug centertemplate_arcgauge %}).

> The ArcGauge is part of the [KendoReact Gauges]({% slug overview_gauges %}) component library. The procedures for installing, importing, and using the Gauges are identical for all components in the package. To learn how to use the ArcGauge and the rest of the Gauges, see the [Getting Started with the KendoReact Gauges]({% slug get_started_gauges %}) guide.

## Key Features

-   [Center template]({% slug centertemplate_arcgauge %})&mdash;You can customize the look of the center of the KendoReact ArcGauge.
-   [Color ranges]({% slug colorranges_arcgauge %})&mdash;Depending on the current value, you can set different colors for the ArcGauge component.
-   [Scale options]({% slug scaleoptions_arcgauge %})&mdash;You can customize the options of the scale in the React ArcGauge component.
-   [Appearance]({% slug styling_gauges %})&mdash;You can quickly style the ArcGauge by choosing from the predefined styling options.
-   [Globalization]({% slug globalization_gauges %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
