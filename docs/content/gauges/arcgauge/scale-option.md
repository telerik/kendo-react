# Scale Options

The ArcGauge enables you to customize the options of the scale.

For example, you can change the start and end angle, the appearance of the label and ticks, customize the ranges, and others. For the full list of options, refer to the [API reference of the ArcGauge]({% slug api_gauges_arcscale %}).
