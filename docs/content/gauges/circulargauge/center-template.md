# Center Template

You can customize the content inside the center of the CircularGauge by using the center template.

To render the center template of a CircularGauge, pass a function to the [`centerRender`]({% slug api_gauges_circulargaugeprops %}#toc-centerRender) property, which accepts value and color arguments.
