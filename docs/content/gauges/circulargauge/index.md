# KendoReact CircularGauge Overview

The KendoReact CircularGauge represents a value by using an arc over a fully enclosed circle.

This gauge type is very similar to the [Arc Gauge]({% slug overview_arcgauge_gauges %}) and supports a similar set of features.

Compared to the [Radial Gauge]({% slug overview_radialgauge_gauges %}), this gauge uses a colored arc instead of an arrow pointer. This leaves the gauge's center free for a user-defined [Center Template]({% slug centertemplate_circulargauge %}).

## Key Features

-   [Center template]({% slug centertemplate_circulargauge %})&mdash;You can customize the look of the center of the KendoReact CircularGauge.
-   [Color ranges]({% slug colorranges_circulargauge %})&mdash;Depending on the current value, you can set different colors for the CircularGauge component.
-   [Scale options]({% slug scaleoptions_circulargauge %})&mdash;You can customize the options of the scale in the React CircularGauge component.
-   [Appearance]({% slug styling_gauges %})&mdash;You can quickly style the CircularGauge by choosing from the predefined styling options.
-   [Globalization]({% slug globalization_gauges %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
