> The LinearGauge is part of the [KendoReact Gauges]({% slug overview_gauges %}) component library. The procedures for installing, importing, and using the Gauges are identical for all components in the package. To learn how to use the LinearGauge and the rest of the Gauges, see the [Getting Started with the KendoReact Gauges]({% slug get_started_gauges %}) guide.

## Key Features

-   [Scale ranges]({% slug scaleranghes_lineargauge %})&mdash;You can customize the scale options in the React LinearGauge component like its range.
-   [Multiple pointers]({% slug multiplepointers_lineargauge %})&mdash;You can highlight multiple values by rendering multiple pointers.
-   [Orientation]({% slug orientation_lineargauge %})&mdash;You can render the LinearGauge either vertically or horizontally.
-   [Scale options]({% slug scaleoptions_lineargauge %})&mdash;You can customize the options of the scale in the React LinearGauge component.
-   [Appearance]({% slug styling_gauges %})&mdash;You can quickly style the LinearGauge by choosing from the predefined styling options.
-   [Globalization]({% slug globalization_gauges %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
