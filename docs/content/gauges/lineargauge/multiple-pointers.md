# Multiple Pointers

The LinearGauge enables you to highlight multiple values by rendering multiple pointers.

To set multiple pointers to a LinearGauge, pass an array of pointers to the [`pointer`]({% slug api_gauges_lineargaugeprops %}#toc-pointer) property.
