# Scale Options

The LinearGauge enables you to customize the options of the scale.

For example, you can change the appearance of the label, line, and ticks, customize the ranges, and others. For the full list of options, refer to the [API reference of the LinearGauge]({% slug api_gauges_linearscale %}).
