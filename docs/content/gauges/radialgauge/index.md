> The RadialGauge is part of the [KendoReact Gauges]({% slug overview_gauges %}) component library. The procedures for installing, importing, and using the Gauges are identical for all components in the package. To learn how to use the RadialGauge and the rest of the Gauges, see the [Getting Started with the KendoReact Gauges]({% slug get_started_gauges %}) guide.

## Key Features

-   [Scale ranges]({% slug scaleranghes_radialgauge %})&mdash;You can customize the range options in the React RadialGauge component.
-   [Multiple pointers]({% slug multiplepointers_radialgauge %})&mdash;You can highlight multiple values by rendering multiple pointers.
-   [Scale options]({% slug scaleoptions_radialgauge %})&mdash;You can customize the options of the scale in the React RadialGauge component.
-   [Appearance]({% slug styling_gauges %})&mdash;ou can quickly style the RadialGauge by choosing from the predefined styling options.
-   [Globalization]({% slug globalization_gauges %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
