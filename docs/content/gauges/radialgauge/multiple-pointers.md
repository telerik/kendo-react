# Multiple Pointers

The RadialGauge enables you to highlight multiple values by rendering multiple pointers.

You can independently customize each pointer including its current value, color, and length. To set multiple pointers to a RadialGauge, pass an array of pointers to the [`pointer`]({% slug api_gauges_radialgaugeprops %}#toc-pointer) property.
