# GridPagerType
The type of the Grid pager.

The available values are:
* `numeric`&mdash;Buttons with numbers.
* `input`&mdash;Input for typing the page number.

```jsx-no-run

<Grid
   pageable="{{
       type: 'numeric'
   }}">
</Grid>
```


`type` GridPagerType = "numeric" | "input";
