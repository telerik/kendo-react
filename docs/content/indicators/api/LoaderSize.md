# LoaderSize
Specifies the size of the Loader
([see example](https://www.telerik.com/kendo-react-ui/components/indicators/loader/appearance#toc-size)).

The possible values are:
* `small`
* `medium` (Default)
* `large`

`type` LoaderSize = "small" | "medium" | "large";
