# KendoReact ColorGradient Overview

The KendoReact ColorGradient renders a gradient, a hue and an alpha slider, and inputs to manually enter the desired color.

The KendoReact ColorGradient component is part of the KendoReact library of React UI components. It is distributed through NPM under the [kendo-react-inputs package](https://www.npmjs.com/package/@progress/kendo-react-inputs).

## Key Features

-   [Disabled ColorGradient]({% slug disabled_colorgradient %})&mdash;You can render a disabled ColorGradient by setting a single property and keep it disabled until certain requirements are met.
-   [Controlled mode]({% slug controlled_colorgradient %})&mdash;You can take advantage of the built-in options that allow you to control the value of the ColorGradient.
-   [Removing opacity inputs]({% slug no_opacity_colorgradient %})&mdash;The ColorGradient supports options that allow the user to specify whether an alpha slider and an alpha input will be rendered.
