# KendoReact MaskedTextBox Overview

The KendoReact MaskedTextBox uses a mask to control the input of the user.

The KendoReact MaskedTextBox component is part of the KendoReact library of React UI components. It is distributed through NPM under the [kendo-react-inputs package](https://www.npmjs.com/package/@progress/kendo-react-inputs).

To learn how to use the MaskedTextBox and the rest of the components in the package, see the [Getting Started with the KendoReact Inputs]({% slug getstarted_inputs %}) guide.

## Key Features

-   [Disabled MaskedTextBox]({% slug disabled_maskedtextbox %})&mdash;You can render a disabled MaskedTextBox and keep it disabled until certain requirements are met.
-   [Read-only MaskedTextBox]({% slug readonly_maskedtextbox %})&mdash;You can override the default active state of the MaskedTextBox.
-   [Valid MaskedTextBox]({% slug valid_maskedtextbox %})&mdash;The built-in validation allows you to ensure that the user provides valid information.
-   [Value]({% slug value_maskedtextbox %})&mdash;The MaskedTextBox provides options for controlling its value.
-   [Masks]({% slug masks_maskedtextbox %})&mdash;You can take advantage of the built-in masks or define your custom rules.
-   [Forms support]({% slug forms_numerictextbox %})&mdash;You can easily integrate the MaskedTextBox with an [HTML5 form](https://developer.mozilla.org/en-US/docs/Learn/HTML/Forms), a [`KendoReact Form`]({% slug overview_form %}), or a third-party React form library.
-   [Globalization]({% slug globalization_inputs %})&mdash;The KendoReact MaskedTextBox comes with globalization features that allow you to create applications that can be used all over the world.
-   [Accessibility]({% slug accessibility_maskedtextbox %})&mdash;The MaskedTextBox is accessible for screen readers and provides full WAI-ARIA support.

> To learn more about the appearance, anatomy, and accessibility of the MaskedTextBox, visit the [Progress Design System documentation](https://www.telerik.com/design-system/docs/components/maskedtextbox/)—an information portal offering rich component usage guidelines, descriptions of the available style variables, and globalization support details.
