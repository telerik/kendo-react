# KendoReact RangeSlider Overview

The KendoReact RangeSlider enables the user to increase, decrease, and select predefined values by dragging its handle along the track or by clicking the value on the scale or using the arrow keys.

The KendoReact RangeSlider component is part of the KendoReact library of React UI components. It is distributed through NPM under the [kendo-react-inputs NPM package](https://www.npmjs.com/package/@progress/kendo-react-inputs).

## Key Features

-   [Labels]({% slug labels_rangeslider %})&mdash;You can use labels to define names for specific positions on the RangeSlider.
-   [Orientation]({% slug orientation_rangeslider %})&mdash;You can render the RangeSlider vertically or horizontally.
-   [Disabled]({% slug disabled_rangeslider %})&mdash;To prevent users from interacting with the RangeSlider, you can render it as disabled.
-   [Controlled mode]({% slug controlled_rangeslider %})&mdash;Take advantage of the built-in options that allow you to control the value of the RangeSlider.
-   [Keyboard navigation]({% slug keyboard_navigation_rangeslider %})&mdash;The RangeSlider support numerous keyboard shortcuts that allow users to interact with the component.
-   [Accessibility]({% slug accessibility_rangeslider %})&mdash;The RangeSlider is accessible for screen readers and provides full WAI-ARIA support.
