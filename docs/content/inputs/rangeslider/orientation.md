# Orientation

The RangeSlider supports a horizontal and a vertical mode of orientation.

By default, the component is in its horizontal mode. To enable the vertical mode, set the [`vertical`]({% slug api_inputs_rangesliderprops %}#toc-vertical) property to `true`.
