> The Signature is part of the [KendoReact Inputs]({% slug overview_inputs %}) component library. The procedures for installing, importing, and using all components in the library are identical. To learn how to use the Signature and the rest of the components in the package, see the [Getting Started with the KendoReact Inputs]({% slug getstarted_inputs %}) guide.

## Key Features

-   [Appearance]({% slug appearance_signature %})&mdash;All KendoReact Inputs enable you to set their appearance settings.
-   [Line Smoothing]({% slug smooth_signature %})&mdash;The Signature can smooth out the user's drawing for better appearance.
-   [Controlled mode]({% slug controlled_signature %})&mdash;Take advantage of the built-in options that allow you to control the value of the Signature.
-   [Disabled Signature]({% slug disabled_signature %})&mdash;You can render a disabled Signature and keep it disabled until certain requirements are met.
