# KendoReact Slider Overview

The KendoReact Slider enables the user to increase, decrease, and select predefined values by dragging its handle along the track or by clicking the side arrow buttons.

The KendoReact Slider component is part of the KendoReact library of React UI components. It is distributed through NPM under the [kendo-react-inputs NPM package](https://www.npmjs.com/package/@progress/kendo-react-inputs).

## Key Features

-   [Labels]({% slug labels_slider %})&mdash;You can use labels to define names for specific positions on the Slider.
-   [Side Buttons]({% slug sidebuttons_slider %})&mdash;The Slider allows you to configure side buttons that increase or decrease the value by a predefined step.
-   [Orientation]({% slug orientation_slider %})&mdash;You can set the Slider's orientation to vertical or horizontal.
-   [Controlled mode]({% slug controlled_slider %})&mdash;Take advantage of the built-in options that allow you to control the value of the Slider.
-   [Keyboard navigation]({% slug keyboard_navigation_slider %})&mdash;The Slider supports numerous keyboard shortcuts that allow users to interact with the component.
-   [Accessibility]({% slug accessibility_slider %})&mdash;The Slider component is accessible for screen readers and provides full WAI-ARIA support.
