# KendoReact Switch Overview

The KendoReact Switch lets the user toggle between checked and unchecked states.

The KendoReact Switch component is part of the KendoReact library of React UI components. It is distributed through NPM under the [kendo-react-inputs package](https://www.npmjs.com/package/@progress/kendo-react-inputs).

## Key Features

-   [Disabled Switch]({% slug disabled_switch %})&mdash;You can render a disabled Switch and keep it disabled until certain requirements are met.
-   [Controlled mode]({% slug controlled_switch %})&mdash;Take advantage of the built-in options that allow you to control the value of the Switch.
-   [Default state]({% slug default_state %})&mdash;The Switch allows you to set its default value.
-   [Labels]({% slug labels_switch %})&mdash;You can set strings as labels for the Switch.
-   [Globalization]({% slug globalization_inputs %})&mdash;The KendoReact Switch comes with globalization features that allow you to create applications that can be used all over the world.
-   [Keyboard navigation]({% slug keyboard_navigation_switch %})&mdash;The Switch allows end-users to control the state of the component with the `space` and `enter` keys.
    support numerous keyboard shortcuts that allow users to interact with the component.
-   [Accessibility]({% slug accessibility_switch %})&mdash;The Switch component is accessible for screen readers and provides full WAI-ARIA support.

> To learn more about the appearance, anatomy, and accessibility of the Switch, visit the [Progress Design System documentation](https://www.telerik.com/design-system/docs/components/switch/)—an information portal offering rich component usage guidelines, descriptions of the available style variables, and globalization support details.
