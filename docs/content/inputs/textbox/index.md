## Key Features

-   [Customization]({% slug adornments_textbox %})&mdash;Provide custom components for `prefix` and `suffix` of the TextBox to achieve advanced scenarios.
-   [Forms support]({% slug forms_support_textbox %})&mdash;You can use the TextBox inside the KendoReact Form component to validate input values, set validation requirements, and prevent the submission of invalid forms.
-   [Floating Label]({% slug floatinglabels_inputs %})&mdash;Add a `Floating Label` functionality to the TextBox component.
-   [Disabled state]({% slug disabled_textbox %})&mdash;You can use the configuration options of the TextBox to disable the component so that users are not able to interact with it.
-   [Appearance]({% slug appearance_textbox %})&mdash;All KendoReact Inputs enable you to set their dimensions.

> To learn more about the appearance, anatomy, and accessibility of the TextBox, visit the [Progress Design System documentation](https://www.telerik.com/design-system/docs/components/textbox/)—an information portal offering rich component usage guidelines, descriptions of the available style variables, and globalization support details.
