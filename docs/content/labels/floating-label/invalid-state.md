# Editors with Invalid State

When the editor has invalid value you can also set different styles for the FloatingLabel through the [`editorValid`]({% slug api_labels_floatinglabelprops %}#toc-editorvalid) property. When set to `false` the color of the FloatingLabel becomes red.
