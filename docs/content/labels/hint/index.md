# Hint Overview

The Hint component enables you to display a hint message to React components.

<CtaPanelOverview></CtaPanelOverview>

The following example shows how to set up the Hint component:

1. Add the Hint component after the desired editor.
1. Set the [`id`]({% slug api_labels_hintprops %}#toc-id) property of the Hint and the `aria-describedby` property of the editor.
