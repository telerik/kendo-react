# KendoReact ActionSheet Overview

The ActionSheet component allows you to display a predefined set of options in a modal view. It is commonly used to present choices related to an action that the user initiates.

## Key Features

-   [Items]({% slug items_actionsheet %})&mdash;The ActionSheet enables you to configure the items based on specific project requirements.
-   [Custom Rendering]({% slug custom_rendering_actionsheet %})&mdash;You can also customize the content of the ActionSheet items.
-   [Animation]({% slug animation_actionsheet %})&mdash;You can add opening and closing animations to the ActionSheet.
-   [Accessibility]({% slug accessibility_actionsheet %})&mdash;The ActionSheet is accessible for screen readers and supports WAI-ARIA attributes.
