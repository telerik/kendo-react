# AppBarPosition
Specifies the AppBar position ([see example](https://www.telerik.com/kendo-react-ui/components/layout/appbar/positioning#toc-position)).

* The possible values are:
* 'top' (Default)
* 'bottom'

`type` AppBarPosition = "top" | "bottom";
