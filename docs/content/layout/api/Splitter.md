# Splitter
Represents the [KendoReact Splitter component](https://www.telerik.com/kendo-react-ui/components/layout/splitter).

```jsx
const App = () => {
   return (
       <div>
           <Splitter
               style={{height: 400}}
               orientation={'vertical'}
           >
               <div>Pane 1</div>
               <div>Pane 2</div>
               <div>Pane 3</div>
           </Splitter>
       </div>
   );
}
```
