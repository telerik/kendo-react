# KendoReact AppBar Overview

The AppBar provides information and actions related to the current application screen. It is typically used to show page titles, brand identity and can contain navigation items.

To learn how to use the AppBar and the rest of the components in the package, see the [Getting Started with the KendoReact Layout Package]({% slug getstarted_layout %}) guide.

## Key Features

-   [Positioning]({% slug positioning_appbar %})&mdash;The built-in options allow you to specify the position of the component relative to the page content.
-   [Content Arrangement]({% slug contentarrangement_appbar %})&mdash;You can easily arrange the layout of the appbar content by using sections, spacings, and separators.
-   [Appearance]({% slug appearance_appbar %})&mdash;The AppBar comes with predefined option that allow you to set its theme color.
-   [Responsive Design]({% slug responsivedesign_appbar %})&mdash;The responsive layout adapts to any device and screen size.
