# Keyboard Navigation


## Managing the Focus


Each item in the bottom navigation is focusable.

## Keyboard Shortcuts

| Shortcut | Behavior |
| -------- | -------- |
| `Shift + Tab` | Focuses previous link. |
| `Tab` | Focuses next link. |
| `Space / Enter` | Activates the currently focused link. |
