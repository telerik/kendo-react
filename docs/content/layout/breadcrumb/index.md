# KendoReact Breadcrumb Overview

The Breadcrumb component allows you to navigate within a folder structure or web page. It provides an easy way to navigate backwards by one or multiple steps. The Breadcrumb is always used in controlled mode.

## Key Features

-   [Item Appearance]({% slug appearance_breadcrumb %})&mdash;The Breadcrumb component comes with predefined options for using font and custom icons.
-   [Delimiter]({% slug delimiter_breadcrumb %})&mdash;You can change the default separator between the Breadcrumb items.
-   [Disabled Mode]({% slug disabled_breadcrumb %})&mdash;You can easily render disabled Breadcrumb items.
-   [Accessibility]({% slug accessibility_breadcrumb %})&mdash;The Breadcrumb is accessible by screen readers and provides full WAI-ARIA support.
