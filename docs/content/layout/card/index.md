# KendoReact Card Overview

The Card component represents any type of content and all kind of actions about a single subject.

It provides clarity, categorization and an attractive way of presenting content.

<CtaPanelOverview></CtaPanelOverview>

The Card component consists of the following building blocks:

-   [`Card`]({% slug api_layout_cardprops %})
-   [`CardHeader`]({% slug api_layout_cardheaderprops %})
-   [`CardTitle`]({% slug api_layout_cardtitleprops %})
-   [`CardSubtitle`]({% slug api_layout_cardsubtitleprops %})
-   [`CardBody`]({% slug api_layout_cardbodyprops %})
-   [`CardActions`]({% slug api_layout_cardactionsprops %})
-   [`CardFooter`]({% slug api_layout_cardfooterprops %})
-   [`CardImage`]({% slug api_layout_cardimageprops %})

## Key Features

-   [Orientation]({% slug orientation_card %})&mdash;You can arrange the content of the Card vertically or horizontally.
-   [Types]({% slug types_card %})&mdash;The predefined types provide a convenient way to change the appearance of the Card.
-   [Actions]({% slug actions_card %})&mdash;The Card allows you to render custom action buttons and to customize their orientation and layout.
-   [Cards Layouts]({% slug layouts_card %})&mdash;You can arrange the Cards in three built-in layouts: list, group and deck.
