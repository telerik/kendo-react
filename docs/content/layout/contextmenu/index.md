# KendoReact ContextMenu Overview

The ContextMenu provides a clean and user-friendly menu that displays a contextual popup with data in a traditional menu-like structure. It lets you invoke commands while preserving screen real estate.

The KendoReact ContextMenu component is part of the KendoReact library of React UI components. It is distributed through NPM under the [kendo-react-layout package](https://www.npmjs.com/package/@progress/kendo-react-layout).
