# Animations

The ExpansionPanel allows you to add expand and collapse animations to it by using the KendoReact Animation components from the [`@progress/kendo-react-animation`]({% slug overview_animation %}) package.

To animate the ExpansionPanel, wrap its content in the desired KendoReact Animation component.
