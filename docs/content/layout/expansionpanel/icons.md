# Custom Icons

The ExpansionPanel provides the option to customize the icon indicators.

You can replace both the expand and collapse icons with custom font icons by using the [`expandSVGIcon`]({% slug api_layout_expansionpanelprops %}#toc-expandsvgicon) and [`collapseSVGIcon`]({% slug api_layout_expansionpanelprops %}#toc-collapsesvgicon) properties.
