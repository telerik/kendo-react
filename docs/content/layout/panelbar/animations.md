# Animations

You can enable or disable the show and hide animations of the PanelBar.

To control the behavior of the animations:

-   Specify the [`animation`]({% slug api_layout_panelbarprops %}#toc-animation) property of the PanelBar.
-   Specify the [`animation`]({% slug api_layout_panelbarprops %}#toc-animation) property to individual children.
