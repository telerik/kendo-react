# PanelBar mounted items

The PanelBar allows its items to stay mounted. To achieve such behavior use the [`keepItemsMounted`]({% slug api_layout_panelbarprops %}#toc-keepItemsMounted) property of the PanelBar and set it to `true`. By default the value is set to `false`.
