# KendoReact StackLayout Overview

The KendoReact StackLayout component allows you to easily align vertically or horizontally multiple elements in a stack.

It delivers different orientations, alignments, spacings and other handy options.

<CtaPanelOverview></CtaPanelOverview>

The following example demonstrates the StackLayout component in action and also showcases other KendoReact components&mdash;the [ButtonGroup]({% slug overview_buttongroup %}), [Avatars]({% slug overview_avatar %}) and [Cards]({% slug overview_card %}) components. The components are arranged in a stack and can be switched from horizontal to vertical orientation.

## Key Features

-   [Orientation]({% slug layout_stacklayout %}#toc-orientation)&mdash;The orientation configuration option allows you to choose between the vertical and horizontal positioning.
-   [Gaps]({% slug layout_stacklayout %}#toc-gaps)&mdash;The Gap property allows you to set gaps between the boxes.
-   [Horizontal alignment]({% slug layout_stacklayout %}#toc-horizontal-alignment)&mdash;You can align the StackLayout horizontally.
-   [Vertical alignment]({% slug layout_stacklayout %}#toc-vertical-alignment)&mdash;You can align the StackLayout vertically.
-   [Nested StackLayouts]({% slug layout_stacklayout %}#toc-nested-stacklayouts)&mdash;You can create complex layouts with nested boxes.
