# Layout

The KendoReact Timeline provides the ability to render events alternatingly on both sides of the axis.

## Alternating Mode

By default, the Timeline renders the events on one side of the axis. To render the events on both sides, set the [`alterMode`]({% slug api_layout_timelineprops %}#toc-alterMode) prop to `true`.
