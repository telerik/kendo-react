# KendoReact ListBox Overview

The KendoReact ListBox is a customizable, responsive, and interactive UI component that enables you to display lists of items that are contained in a box and allow selecting, reordering, deleting, as well as the dragging and dropping of items.

## What is the KendoReact ListBox?

The ListBox is a native KendoReact component built specifically for the React ecosystem and distributed via npm.

> [@progress/kendo-react-listbox](https://www.npmjs.com/package/@progress/kendo-react-listbox)

## Key Features

The KendoReact ListBox provides the following key features. Note that this is not an exhaustive list.

-   Highly customizable and easy to style **appearance**
    -   Built-in appearance options to adjust the layout and the dimensions of your listbox
    -   [Built-in themes](slug://themesandstyles)
    -   [Theme customization](slug://customizingthemes)
    -   [Custom themes](slug://themebuilder)
-   [Selection](slug://selection_listbox)
-   [Drag and Drop](slug://draganddrop_listbox)
-   [Customization of the ListBox items](slug://customization_listbox)
-   [Multiple Lists](slug://multiplelists_listbox)

## How Does the KendoReact ListBox Work?

The KendoReact ListBox is built natively for React, with no dependencies. It leverages key React concepts such as components-based architecture, props and state management, conditional rendering, and event handling.

You have complete control over your React listbox and you can extend or override its default behaviors and appearance via props and event handlers.

## Frequently Asked Questions

### Is the KendoReact ListBox Free to Use

Yes. The KendoReact ListBox is free to use without any license or registration with Progress or Telerik.

### How to Create ListBox in React

To create a listbox in React, you need to install the KendoReact ListBox package and then import and configure the listbox component in your application. For more information, refer to the [Getting Started with the KendoReact ListBox](slug://get_started_listbox) article.
