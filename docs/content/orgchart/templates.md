# KendoReact OrgChart Templates

The OrgChart allows customizing the of the card data using the [`itemRender`]({% slug api_orgchart_orgchartprops %}#toc-itemRender) property as well as the group title and subtitle using the [`groupTitleRender`]({% slug api_orgchart_orgchartprops %}#toc-groupTitleRender) and [`groupSubtitleRender`]({% slug api_orgchart_orgchartprops %}#toc-groupSubtitleRender) properties respectively.
