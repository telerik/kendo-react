# Scroll to page

The KendoReact PDF Viewer component allows scrolling to a specific page programmatically using its [scrollToPage]({% slug api_pdf-viewer_pdfviewerprops %}#scrollToPage) function.
It accepts the root HTML element of the PDF Viewer and the page number to which you want to scroll to.
