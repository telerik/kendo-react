# PivotGrid Configurator Overview

The KendoReact `PivotGridConfigurator` component represents an OLAP cube structure.

The `PivotConfigurator` can be used for configuring the column and row dimensions, and the measure fields of the `PivotGrid`. It is a supplementary component which is strongly related to the KendoReact `PivotGrid` component.
