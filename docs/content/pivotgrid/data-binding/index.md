# Data Binding

The KendoReact PivotGrid supports data binding to an Online Analytical Processing (OLAP) cube that is accessible through HTTP and data binding to local data.

## OLAP Service

KendoReact provides an OLAP service dll that you can use for testing. It is hosted at `https://demos.telerik.com/service/v2/olap/msmdpump.dll`. To see the responses, you must query the service, you cannot open it directly in the browser. For more information on binding the PivotGrid to data over an OLAP cube, refer to the following articles:

-   [OLAP Cube Fundamentals]({% slug data-binding_olap-fundamentals_pivotgrid %})
-   [OLAP Cube Setup]({% slug data-binding_olap-fundamentals_pivotgrid %})
-   [Configurator Overview]({% slug configurator_pivotgrid %})

## Local Data

The KendoReact PivotGrid component enables you to data-bind to local data. See the [Local Data Binding]({% slug localbinding_pivotgrid %}) article.
