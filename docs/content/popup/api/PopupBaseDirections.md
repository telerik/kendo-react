# PopupBaseDirections
The possible base directions for slide and push animations.

Possible values: 'left', 'right', 'up', 'down'.

```ts
<Popup animation={{ direction: 'left' }} />
```


`type` PopupBaseDirections = "left" | "right" | "up" | "down";
