# PopupExpandDirection
The possible directions for expand animations.

Possible values: 'horizontal', 'vertical'.

```ts
<Popup animation={{ type: 'expand', direction: 'vertical' }} />
```


`type` PopupExpandDirection = "horizontal" | "vertical";
