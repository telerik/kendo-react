## What is the KendoReact Popup?

The Popup is a native KendoReact component built specifically for the React ecosystem and distributed via npm.

> [@progress/kendo-react-popup](https://www.npmjs.com/package/@progress/kendo-react-popup)

### Key Features

The KendoReact Popup provides the following key features. Note that this is not an exhaustive list.

-   Highly customizable and easy to style **appearance**
    -   [Built-in themes](slug://themesandstyles)
    -   [Theme customization](slug://customizingthemes)
    -   [Custom themes](slug://themebuilder)
-   Customizable [Animations](slug://animations_popup)
-   Full control over the [Aligning and Positioning](slug://alignmentpositioning_popup)
-   Ability to modify all Popup components by using the `PopupPropsContext` component.
-   [Boundary Detection](slug://viewportboundarydetection_popup)
-   Out-of-the-box **accessibility** and **keyboard navigation**

## How Does the KendoReact Popup Work?

> The KendoReact Popup provides [free React popup](#package-contents) that don't require a commercial license key or an active trial license key to install or use.

The KendoReact Popup is built natively for React, with no dependencies. They leverage key React concepts such as components-based architecture, props and state management, conditional rendering, and event handling.

You have complete control over your React popup and you can extend or override their default behaviors and appearance via props and event handlers.

### React Popup with Other KendoReact Components

You can combine multiple KendoReact components with the React popup in the same application:

-   Display confirmation Popup in a [Grid](slug://overview_grid) for row actions
-   Embed [Form](slug://overview_form) inside a Popup

> The Popup package is a dependency for multiple other KendoReact components.

### React Popup with Other Telerik Components

Based on the frameworks and technologies used in your project, you can combine UI components from different Telerik and Progress suites.

## Frequently Asked Questions

### Is the KendoReact Popup Free to Use?

Yes. The KendoReact Popup is free to use without any license or registration with Progress or Telerik.

### How to Create a Popup in React?

To create a popup in React, you need to install the KendoReact Popup package and then import the component in your application. For more information, refer to the [Getting Started with the KendoReact Popup](slug://get_started_popup) article.
