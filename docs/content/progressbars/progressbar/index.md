# KendoReact ProgressBar Overview

The ProgressBar displays and tracks the progress of a task.

It supports different label formats, horizontal and vertical orientation, and also different directions.

> The ProgressBar is part of the [KendoReact ProgressBars]({% slug overview_progressbars %}) component library. The procedures for installing, importing, and using the ProgressBars are identical for all components in the package. To learn how to use the ProgressBar and the rest of the ProgressBars, see the [Getting Started with the KendoReact ProgressBars]({% slug get_started_progressbars %}) guide.

## Key Features

-   [Value and ranges]({% slug progressbar_value_ranges %})&mdash;You can set the value and the value range of the ProgressBar component.
-   [Disabled ProgressBar]({% slug progressbar_disabled %})&mdash;You can disable the KendoReact ProgressBar component.
-   [Orientation]({% slug progressbar_orientation %})&mdash;You can set the orientation to `horizontal` or `vertical`.
-   [Direction]({% slug progressbar_direction %})&mdash;You can reserve the direction of the ProgressBar component.
-   [Custom label]({% slug progressbar_label %})&mdash;You can add custom labels to the ProgressBar component.
-   [Animation]({% slug progressbar_animation %})&mdash;You can animate the ProgressBar when the value is changing.
-   [Appearance]({% slug progressbar_appearance %})&mdash;You can quickly style the ProgressBar by choosing from the predefined styling options.
-   [Globalization]({% slug rtl_progressbars %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
