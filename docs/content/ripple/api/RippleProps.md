# RippleProps
Represents the KendoReact Ripple component.

The ripple effect will show on the following elements:
- Buttons
- Checkboxes
- Radio buttons

```jsx-no-run
<Ripple disabled="true">
     <Button> Default Button</Button>
</Ripple>
```
