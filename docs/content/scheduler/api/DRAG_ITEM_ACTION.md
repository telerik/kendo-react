# DRAG_ITEM_ACTION
Represents the available `action` types for the `SchedulerItemDragItemAction` objects.

The available types are:
- `DRAG_ITEM_SET'`
- `DRAG_ITEM_START'`
- `DRAG_ITEM_DRAG'`
- `DRAG_ITEM_COMPLETE'`
- `DRAG_ITEM_COMPLETE_OCCURRENCE'`
- `DRAG_ITEM_COMPLETE_SERIES'`
- `DRAG_ITEM_RESET'`
- `DRAG_ITEM_DRAG_SELECTED`
