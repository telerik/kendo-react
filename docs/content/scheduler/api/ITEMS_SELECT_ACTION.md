# ITEMS_SELECT_ACTION
Represents the available `action` types for the `SchedulerItemsSelectAction` objects.

The available types are:
- 'ITEMS_SELECT_SELECT`
- 'ITEMS_SELECT_SELECT_NEXT`
- 'ITEMS_SELECT_SELECT_PREV`
- 'ITEMS_SELECT_RESET`
- 'ITEMS_SELECT_ADD`
- 'ITEMS_SELECT_REMOVE`
