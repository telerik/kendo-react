# REMOVE_ITEM_ACTION
Represents the available `action` types for the `SchedulerItemRemoveItemAction` objects.

The available types are:
- `REMOVE_ITEM_SET`
- `REMOVE_ITEM_RESET`
- `REMOVE_ITEM_COMPLETE`
