# SLOTS_FOCUS_ACTION
Represents the available `action` types for the `SchedulerSlotsFocusAction` objects.

The available actions are:
- `SLOT_FOCUS_UP`
- `SLOT_FOCUS_DOWN`
- `SLOT_FOCUS_LEFT`
- `SLOT_FOCUS_RIGHT`
