# SLOTS_SELECT_ACTION
Represents the available `action` types for the `SchedulerSlotsSelectAction` objects.

The available actions are:
- `SLOTS_SELECT_SELECT`
- `SLOTS_SELECT_RESET`
- `SLOTS_SELECT_ADD`
