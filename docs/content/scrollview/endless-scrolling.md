# Endless Scrolling

The ScrollView provides a scrolling mode which loops through its views in an endless fashion.

To enable or disable the endless scrolling functionality, set the [`endless`]({% slug api_scrollview_scrollviewprops %}#toc-endless) property. By default, `endless` is set to `false`.
