# KendoReact Tooltip Overview

he KendoReact Tooltip displays informative text when the user clicks or hovers over an element.

The KendoReact Tooltip component is part of the KendoReact library of React UI components. It is distributed through NPM under the [kendo-react-tooltips package](https://www.npmjs.com/package/@progress/kendo-react-tooltip).

## Functionality and Features

-   [Controlled mode]({% slug controlledmode_tooltip %}) - The KendoReact Tooltip provides options for you to show it programmatically not only on hover.
-   [Specifying anchor elements]({% slug filter_elements_tooltip %}) - You can target specific anchor elements which will render a tooltip.
-   [Defining the position]({% slug positioning_tooltip %}) - By default, the position of the Tooltip is determined by the available space around it. You can also specify other positions.
-   [Setting a content template]({% slug templates_tooltip %}) - The KendoReact Tooltip allow setting a custom rendering for its content.
-   [Rendering multiple Tooltips]({% slug multiple_tooltips_tooltip %}) - You can display multiple Tooltips on the same page.
-   [Customizing the Tooltip]({% slug customization_tooltip %}) - You can place the Tooltip in a specific location and attach its handlers there.
