# Keyboard Navigation


## Managing the Focus


The Tooltip is not focusable and does not contain focusable elements.

## Keyboard Shortcuts


When the element which triggered the tooltip is focused, `Escape` dismisses the tooltip.

| Shortcut | Behavior |
| -------- | -------- |
| `Escape` | Dismisses the Tooltip. |
