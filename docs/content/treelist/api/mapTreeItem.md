# mapTreeItem
Creates a new array with the results of calling the provided callback function
on the element which match the `level` in the provided data tree.




#### Parameters
##### tree
<code>


any[]


</code>
The data tree.

##### level
<code>


number[]


</code>
An array of indexes of each parent and current item in the data tree.

##### subItemsField
<code>


string


</code>
The field which points to the subitems collection of each data item.

##### callback
<code>


(value: any) => any


</code>
The callback function.
