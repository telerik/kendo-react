# Expanding Items

To render an expand icon next to each parent item, set the [`expandIcons`]({% slug api_treeview_treeviewprops %}#toc-expandicons) property of the TreeView to `true`.

A TreeView item is expanded when its `expanded` field is set to `true`. Instead of `expanded`, you can implement your own field by setting the [`expandField`]({% slug api_treeview_treeviewprops %}#toc-expandfield) property to the name of that custom field.
