# Rendering Changes in 13.0.0

The KendoReact 13.0.0 release introduces changes in the rendering of several components.

## Chat

-   Removed `k-avatars` class from the `.k-message-list` element.
-   `.k-typing-indicator` is now nested under `.k-chat-bubble.k-bubble > .k-bubble-content` instead of `.k-chat-bubble.k-bubble`.

## Tabstrip

-   Removed `k-item` class.

## Treelist & Gantt

-   Replaced the `td` elements inside the filter row with `th`.
-   Replaced the `k-table-td k-header` class of the header filter elements with `k-table-th k-header`.
