# Rendering Changes in 5.18.0

The KendoReact 5.18.0 release introduces changes in the rendering of several components. 

## DateInputs
  * Removed the `k-calendar-content` class from the `k-content` elements.

## TimePicker
  * Removed the `k-time-part` class from the `TimePicker` popup `<div>` wrapper.
