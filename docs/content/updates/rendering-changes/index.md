# List of KendoReact Releases Introducing Rendering Changes

As the KendoReact library constantly evolves, the framework may introduce changes in the styling and rendering configurations of its components with time.

These changes will affect the appearance of the components if, as a result, your business application turns out to have overridden the built-in CSS rules.

The following list provides the KendoReact releases which introduce changes in the display of the available components:
