# toText
Converts an unknown value to a human-readable string suitable for
returning in tool results. Primitive values are stringified directly;
objects are JSON-serialised; non-serialisable objects fall back to '[object]'.

#### Parameters
##### value
<code>


unknown


</code>


#### Returns
<code>


string


</code>
